# 🚆 Premium Train Tracker

**Designed and developed by Gautam Industries**

Premium Train Tracker is a sleek, modern, and highly responsive web application that allows users to seamlessly track Indian Railways trains in real-time. It features a beautiful dark theme, intuitive UI, and precise live tracking data.

## ✨ Features

- **Spot Your Train (Live Tracking)**: Get the real-time exact location of any running train, complete with a beautifully animated timeline layout.
- **Live Station Board**: View all upcoming and departing trains at any station in the next 2-8 hours.
- **Train Schedule**: Look up the full route and timetable of any specific train.
- **Trains Between Stations**: Effortlessly find a list of trains running between any two stations.
- **Premium UI/UX**: Designed from the ground up with modern aesthetics, smooth hover effects, clear typography, and a Vercel-ready architecture.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Custom Dark & Light themes), JavaScript (ES6+), FontAwesome for icons.
- **Backend**: Node.js, Express.js
- **APIs Used**: Custom scraping algorithms and integration with various live railway APIs (e.g., Rappid.in, erail.in).
- **Deployment**: Configured for 1-click deployment to **Vercel**.

## 🚀 Getting Started Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/devilisonline/Train_tracker_Website.git
   cd Train_tracker_Website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the local server**:
   ```bash
   node index.js
   ```

4. **Access the application**:
   Open your browser and navigate to: `http://localhost:8080`

## 🌍 Deploying to Vercel

This project is already pre-configured to be deployed natively on Vercel.

If you have the Vercel CLI installed, simply run:
```bash
npx vercel --prod
```
Vercel will automatically detect the configuration in `vercel.json` and deploy your app globally in seconds.
